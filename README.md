# MSCapp - Microsoft Club SIST

A simple Android application built for the Microsoft Club SIST Week 4 task.

## App Flow

Splash Screen → Profile Setup → Home Screen

## Features

- Splash screen with Microsoft Club logo (3 second timer)
- Profile setup screen with photo picker and name input
- Home screen with personalized welcome message
- Clean and simple UI with Microsoft blue theme

## Tech Stack

- Language: Kotlin
- IDE: Android Studio
- Min SDK: 24
- Target SDK: 36

## How to Run

1. Clone the repository
2. Open in Android Studio
3. Add your own `microsoft_club_logo.png` to `res/drawable`
4. Run on emulator or physical device
